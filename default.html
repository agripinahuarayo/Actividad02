Diseñar sitios web teniendo en cuenta la experiencia del usuario.

Si los visitantes encuentran que su sitio web es fácil de navegar, probablemente volverán y la próxima vez también se lo contarán a sus amigos. El diseño UX es el proceso de diseñar un sitio web que sea útil, fácil de usar y agradable para interactuar. La UX también es un factor muy importante a considerar para la optimización de motores de búsqueda (SEO).
